Welcome to Plotter

How to use

  • Plotter: Enter a source and target system to find the shortest path. Enable "Neutron Highway" for faster travel using neutron stars boosting (especially useful with Caspian Explorer Mk II Supercharge Optimised Frame Shift Drive 6x boost). Regular plotting with 500 ly max hop can be used for carrier plotting.
  • Search: Find the nearest systems from a specific location, filtered by star types.
  • Library: Save your favorite routes for quick access later. All data is stored locally in your browser.

Technical Details

The backend is powered by a custom SQLite implementation utilizing prefix-compressed system names for fast lookups. To maintain high performance while keeping costs low:

  • Cloud Hosting: The system runs on Azure Container Apps with auto-shutdown enabled.
  • Backend Warmup: If the app hasn't been used recently, it may take about a minute to start the backend service. A progress bar will appear during this time.
  • Dataset Optimization: To reduce the ~190M system dataset, the database is filtered:
    • Unique/hand-named systems and non-common star types (Neutron, Black Holes, etc.) are always preserved.
    • Common stars are down-sampled to one system per 50×50×50 ly cube to maintain spatial coverage while reducing density.
    • This results in approximately 2GB database size.
  • Thanks: Elite Dangerous community, especially EDSM and Spansh and of course fdev.
