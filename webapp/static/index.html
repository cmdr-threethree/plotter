Plotter

Welcome to Plotter

Plotter is a high-performance navigation tool designed for deep-space pathfinding. It uses a specialized A* implementation to find efficient routes between star systems.

How to use

  • Plotter: Enter a source and target system to find the shortest path. Enable "Neutron Highway" for faster travel using neutron stars.
  • Search: Find the nearest systems from a specific location, filtered by star types.
  • Library: Save your favorite routes for quick access later. All data is stored locally in your browser.

Technical Details

The backend is powered by a custom SQLite implementation utilizing prefix-compressed system names for ultra-fast lookups. To maintain high performance while keeping costs low:

  • Cloud Hosting: The system runs on Azure Container Apps with auto-shutdown enabled.
  • Backend Warmup: If the app hasn't been used recently, it may take about a minute to start the backend services. A progress bar will appear during this time.
  • Filtering: "Find Nearest" uses a spatial cube filter (side length of 50 ly) to efficiently narrow down candidate systems before calculating exact distances.
